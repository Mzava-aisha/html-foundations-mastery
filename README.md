# HTML5 Practice Tasks

This repository contains exercises for learning HTML5, following a structured learning path from basic HTML skeleton to advanced elements like forms, tables, and SVG.

## Tasks Completed

1. **HTML Skeleton**
   - Created a basic HTML5 page with `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>`.

2. **Text Content**
   - Added headings (`<h1>`–`<h6>`) and paragraphs (`<p>`).

3. **Lists**
   - Used `<ul>` for unordered lists and `<ol>` for ordered lists.

4. **Media**
   - Inserted images using `<img>`.

5. **Links**
   - Added hyperlinks with `<a>`.

6. **Semantic HTML Structure**
   - Used `<header>`, `<nav>`, `<main>`, and `<footer>`.

7. **Tables**
   - Created simple tables using `<table>`, `<tr>`, `<th>`, `<td>`.
   - Used `colspan` and `rowspan` to merge cells.

8. **Forms**
   - Created basic forms with `<input>` and `<label>`.
   - Added dropdowns (`<select>`), radio buttons, and `<textarea>`.
   - Applied validation with `required`, `minlength`, and `pattern`.
   - Grouped elements using `<fieldset>` and `<legend>`.

9. **Meta Tags**
   - Added `<meta>` for `description` and `viewport` (responsive design).

10. **SVG**
    - Created inline SVG with a circle and a rectangle.

## How to Use

- Open the HTML files in a browser to see the results.
- Forms include basic HTML5 validation.
- Tables and SVG demonstrate proper HTML structure and attributes.
